Inheritence:
=============

Getting the data from parent class to child class called as inheritence.

we will implement inheritence with the help of the extends keyword.

single level inheritence
multi level inheritence
multiple inheritence
Hirearichal inheritence
Hybrid inheritence

single leve1 inheritence:
=========================

class Parent{
    var_one:string = "hello";
}

class Child extends Parent{

    var_two:string = "hello";
    var_three:string ="hai"

}

let object:Child= new Child();
console.log(object.var_one,object.var_two,object.var_three);

let p:Child = new Child();

console.log(p.var_one,p.var_two,p.var_three);

let c:Parent = new Child();
console.log(c.var_one);

let p1:Child = new Parent();
console.log(p1.var_one,p1.var_two,p1.var_three);//child class reference unable to hold parent class object.


This is single level inheritence.


Multilevel Inheritence:
=======================

parent ->child -> sub child

===========================================================

class Parentmulti{
    fun_one():string{
        return "This is parent class";
    }
}

class Child1 extends Parentmulti{

    fun_two():string{
        return "This is child class";
    }
}

class subchild1 extends Child1{

    fun_three():string{
        return "This is subchild class";
    }
}

let fu:subchild1 = new subchild1();
console.log(fu.fun_one(),fu.fun_two(),fu.fun_three());

let su:Child1 = new Child1();
console.log(su.fun_one(),su.fun_two());

let pa:Parentmulti = new Parentmulti();
console.log(pa.fun_one());


In multilevel inheritence we have only one parent and one child and one sub child.

Abstract classes:
=================
if we know partial implementations then we will choose abstract classes.

Abstract is the keyword used to create the abstract classes.

Implementations provided by child classes.

Example:

			abstract class Class_one{

				fun_one():string{

					return "function one";
				}

				abstract fun_two():string;

			}

			abstract class Class_two extends Class_one{

				fun_two(): string {
					return "function_two"
				}

			   abstract fun_three():string;
			}

			class Class3 extends Class_two{

				fun_three(): string {
					return "sravs";
				}
			}

			let obj:Class3 = new Class3();

			console.log(obj.fun_one(),obj.fun_two(),obj.fun_three());
			
Note: We can't create object to the abstract classes as well as interfaces also.

	interface inter1{
